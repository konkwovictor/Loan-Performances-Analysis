# 📊 Loan Performance & Customer Risk Analysis Dashboard

## 🚀 Project Overview
This project analyzes loan performance and customer credit risk using interactive Power BI dashboards.

The goal is to transform raw, messy financial data into actionable insights that help financial institutions make better lending decisions.

---

## 🎯 Business Problem
Financial institutions struggle to:
- Track loan performance across regions
- Identify high-risk customers early
- Understand credit distribution across income groups
- Monitor loan trends over time

This dashboard provides a centralized view to solve these challenges.

---

## 📌 Objectives
- Analyze total loan and credit distribution
- Track monthly loan trends
- Identify high-risk customers
- Segment customers by income group
- Visualize geographic loan distribution

---

## 📂 Dataset Description
The dataset contains:
- Customer ID  
- Loan Amount  
- Credit Score  
- Income Group  
- Interest Rate  
- Country  
- Loan Type  
- Date  

### ⚠️ Data Issues
- Missing values  
- Inconsistent category labels  
- Outliers  

---

## 🧹 Data Cleaning (Power Query)
Steps performed:
- Replaced and handled null values  
- Standardized categorical fields  
- Removed extreme outliers  
- Created derived columns (income group, segments)  
- Fixed data types (dates, numeric values)  

---

## 📐 Data Modeling
- Created a Calendar Table for time analysis  
- Built relationships between tables  
- Enabled time intelligence calculations  

---

## 📊 Dashboard 1: Loan Performance Overview

### KPIs
- Total Loan Amount  
- Total Credit Amount  
- Total Annual Revenue  
- Total Customers  

### Key Insights
- Monthly trends reveal seasonal loan patterns  
- Certain loan types drive higher revenue  
- Geographic regions show uneven loan distribution  

---

## 📊 Dashboard 2: Customer Risk & Credit Analysis

### KPIs
- Average Loan Amount  
- Average Credit Score  
- Average Interest Rate  
- Average Age  

### Key Insights
- High-income customers generally have better credit scores  
- Some countries show higher credit risk levels  
- Credit distribution highlights risk clusters  



---

## 📈 Key Findings
- High loan concentration in specific regions  
- Middle-income group drives most loan volume  
- Certain segments show higher default risk  
- Loan issuance varies seasonally
